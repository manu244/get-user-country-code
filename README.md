# get-user-country-code
Get Country code with moment js
 * these both libraries are needed to get the response
        npm install moment
        npm install moment-timezone 
 * Countries time zone json file needed


response :-  
     {
        timezones: ["Asia/Calcutta"],
        latlng: [20, 77],
        name: "India",
        country_code: "IN",
        capital: "New Delhi",
    };
